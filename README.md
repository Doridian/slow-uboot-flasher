# slow-uboot-flasher

- `npm i serialport`
- Adjust COMPORT and FILENAME in file
- `node ./flashme.js`
- Plug it in once the app tells you to, not before. This is so it can cancel autoboot.
- Wait a few hours (literally, see below for how it looks while in progress)
